# Северный Технопарк — PWA v5 Corporate Edition

Готово для GitHub Pages.

## Как загрузить на GitHub Pages
Загрузите содержимое папки `severny_app_v5_corporate` в корень репозитория.

В корне должны лежать:
- index.html
- manifest.json
- sw.js
- css/
- js/
- assets/

## Если видна старая версия
Откройте сайт с параметром:
?v=5

Или очистите кэш браузера/удалите старую PWA с телефона и добавьте заново.
